# Deep Agents from Scratch - CV Project Description

## Professional Summary (100 words)

Built an educational framework demonstrating advanced AI agent architectures using LangGraph and Claude AI. Implemented production-grade context engineering patterns including task planning with TODO lists, virtual file systems for context offloading, and hierarchical sub-agent delegation for complex reasoning tasks. The project features a progressive tutorial series (5 Jupyter notebooks) teaching ReAct loops, state management with Pydantic, and web search integration via Tavily API. Designed reusable components for building research agents capable of multi-step reasoning, similar to commercial systems like Anthropic's Claude Code. Technologies: Python 3.11+, LangGraph, LangChain, Anthropic Claude API, asynchronous programming, and state-driven agent orchestration.
